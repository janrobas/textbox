# Textbox

A simple notepad application written in Lazarus/Free Pascal. It's a basic text editor without unnecessary complexity. Okay for viewing simple text files, editing hosts file and stuff like that - not really for editing code.

![Textbox](screenshot.png)

## What it does

- Edit plain text files (only UTF-8!)
- Open and save files with UTF-8
- Dark/light theme
- Word wrap option
- Font customization
- Find text in documents

## What it doesn't do

This is intentionally minimal. Don't expect:
- Syntax highlighting
- Tabs/multiple documents
- Advanced formatting
- Plugins or extensions
- Copilot

## Getting started

### Prerequisites
- Lazarus IDE (tested with 4.2), Free Pascal 3.2.2

### Building
1. Open `textbox.lpi` in Lazarus
2. Run -> Build
